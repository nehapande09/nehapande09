## featured projects
 
### 🌿 YouMatter — *you are not alone. you never were.*
> A calm, minimal app for overthinkers. Write freely, breathe through it, release what's heavy.
 
**What it does:** Daily diary with AI mood detection · 3 breathing patterns with animated guide · Emotion release page (words vanish on release) · Panic attack grounding guide · AI book recommendation based on your mood · 4 soothing themes
 
**Built with:** React 18 · Vite · Tailwind CSS · Context API · useReducer · Custom Hooks · Claude API · localStorage · GitHub Pages

### 🔍 VeriFact — *because the internet lies*
> AI-powered fact-checking platform that verifies claims using RAG and real-time web sources.
 
**What it does:** Submit any claim → AI cross-references multiple sources → returns a verdict with evidence and confidence score · Source citation with links · Handles nuanced claims, not just yes/no
 
**Built with:** Python · Flask · React · LangChain · RAG (Retrieval-Augmented Generation) · Claude API · REST API
